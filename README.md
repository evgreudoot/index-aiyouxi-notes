# index-aiyouxi-notes

一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定网站或域名，仅作为静态页面集合的托管与索引。

## 项目简介

本仓库存储了一系列独立的 HTML 文件，这些页面可能涉及笔记、工具、演示或其它用途。每个页面均为自包含的静态文档，不依赖外部服务或数据库。仓库旨在提供一个整洁、可访问的页面归档空间。

## 目录说明

```
/
├── index.html          # 主页（页面索引）
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 可选：共享资源（CSS、JS、图片等）
    └── ...
```

- `index.html`：仓库主页，通常包含页面列表或导航。
- `pages/`：核心目录，所有独立 HTML 文件均放置于此。
- `assets/`：可选目录，用于存放多个页面共用的样式、脚本或媒体文件。

## 页面归档说明

- 每个 HTML 文件均为独立文档，可单独访问。
- 页面内容不包含外部链接或第三方资源（除非在 `assets/` 中托管）。
- 页面命名应保持清晰、一致，建议使用小写字母与连字符。
- 若页面需要引用共享资源，请使用相对路径，如 `../assets/style.css`。

## 维护说明

- 本仓库由维护者定期更新，新增或修改页面时请遵循上述目录结构。
- 不建议在仓库中存放大型二进制文件（如视频、压缩包），以保持仓库轻量。
- 欢迎提交 Issue 或 Pull Request 讨论页面归档相关事宜。
- 仓库不用于推广、营销或商业用途，仅作为个人或协作的静态页面集合。
